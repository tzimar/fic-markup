This is an example of this formatting system.

Italics: *like this*.
Bold: **like this**.
Bold italics: ***like this***.
Underlined: _like this_.
Struckthrough: ~like this~.
Small caps: $like this$.

Elements can be set like this:

[center]
  This text will be centered.
[/center]

Elements can specify their ID and class:

[div.text-messages#from-gf]
  GG: hiii guys!!!
  GG: whats up?
  TG: not much
[/div]

Elements can appear inline: [i]like this[/i].

Section breaks:

This is a paragraph in chapter 1.
~~~
[center]This is a paragraph in chapter 1.[/center]
~~~~~~~~~~~~~~~
This is a paragraph in chapter 1.


This is a paragraph in chapter 1.
>>>
[center]This is a paragraph in chapter 1.[/center]
>>>
This is a paragraph in chapter 1.

Narration.

= Dialog.

Narration. = Dialog.

= Dialog. = Narration.

Narration. = Dialog. = Narration.

= Dialog. = Narration. = Dialog.

Comments can be specified like this:

#[ this won't appear in the final document at all! ]#

Comments can even appear #[within text]# like this!

A class or ID can be applied to the next element with some tag:

[>p.not-inset]
[center]
  This text won't be inset.
[/center]
