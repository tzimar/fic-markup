This is a markup language combining aspects of BBCode and Markdown, designed for writing fanfics in a readable format while not chaining the output too much to things like whitespace.

It renders to HTML, and is currently required to be styled with a specific set of classes, but this may change to allow greater flexibility.

Input-output example pairs are provided in the `examples` directory. Run the `test.py` script to ensure that `render.py` produces correct output.
